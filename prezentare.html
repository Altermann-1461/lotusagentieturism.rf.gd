Prezentare Aplicație Web: Gestionare Oferte Turistice

1. Descrierea Aplicației

Aplicația permite gestionarea ofertelor turistice. Fiecare ofertă conține patru atribute:

  • ID – Identificator unic, autoincrementat
  • Țara – Destinația turistică
  • Preț – Costul ofertei în Euro
  • Număr de zile – Durata sejurului

În prezent, oricine poate accesa și gestiona ofertele, dar se va implementa un sistem de autentificare.

2. Arhitectura Aplicației

Aplicația urmează arhitectura **Model-View-Controller (MVC)** și conține următoarele componente:

  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP, MySQL
  • Baza de date: Stochează ofertele

Principalele operațiuni:

  • Adăugare ofertă (Create)
  • Vizualizare oferte (Read)
  • Modificare ofertă (Update)
  • Ștergere ofertă (Delete)

3. Structura Bazei de Date

            CREATE TABLE oferte (
                id INT AUTO_INCREMENT PRIMARY KEY,
                tara VARCHAR(100) ,
                pret INT ,
                zile INT
            );

Tabela oferte permite stocarea detaliilor fiecărei oferte turistice.

4. Soluția de Implementare

Implementarea va folosi PHP pentru backend, iar datele vor fi gestionate într-o bază de date MySQL.

Vom folosi interogări pregătite (`Prepared Statements`) pentru securitate și protecție împotriva SQL Injection.
